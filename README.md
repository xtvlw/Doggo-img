# Doggo Images

this web page get random images of dogs.

## dependences

- Node.js => 16v
- npm => 8v

### how to set up

the code bellow will install any dependences for the app and start the project in `http://localhost:3000/`

```bash
git clone https://github.com/xtvlw/Doggo-img.git
cd Doggo-img
npm install
npm start
```
