# Doggo Images

This web page get random images of dogs.

## Was build with
- React.JS
- CSS
- Love ❤️

## Dependencies
- Node.js => 16v
- NPM => 8v

## how to set up
the code below will install any dependence for the app and start the project in `http://localhost:3000/`

```bash
git clone https://github.com/xtvlw/Doggo-img.git
cd Doggo-img
npm install
npm start
```
